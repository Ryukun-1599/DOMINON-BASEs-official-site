拠点リスト · ゲーム内通貨表示

DonutSMP で使われる K / M / B / T の感覚に合わせた価格表記です。ここでの購入ボタンは Discord Webhook への購入依頼通知であり、ゲーム内の即時決済ではありません。

Access policy

Minecraft: 未登録 Discord: 未使用

身元の登録（購入前に必須）

Minecraft ユーザー名は必須です。Discordはサーバ方針に応じて必須にできます（管理画面で切替）。OAuth が使えない場合は、手入力欄に Discord 表示名を入れてください。

Discord で連携 は Developer Portal でこのサイトの callback.html を Redirect に追加し、必要なら Implicit フローを有効化してください。

公開中の拠点

価格は目安（掲載者設定）

取引の流れ（意味のあるボタン設計）

  1. 身元情報を保存 — 取引連絡に使う Minecraft 名（＋Discord）をこのブラウザに記録します。
  2. 拠点仕様を確認 — 区域・座標・注意事項を読み、合意できるか判断します。
  3. 購入依頼を送る — 運営の Discord Webhook に「商品 / ユーザー / 日時」形式で通知します（確定ではありません）。
DOMINIO BASEs — Minecraft を題材にしたファン用途の静的ページです。Mojang / Microsoft および DonutSMP 運営とは無関係です。
通知に失敗する場合は、ブラウザ設定・Discord アプリ設定（Redirect / OAuth）・Webhook URL を確認してください。
